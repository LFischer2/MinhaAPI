# 🩸 Sistema de Doação de Sangue — Backend API

## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento do backend de um sistema de doação de sangue utilizando:

* C# (.NET / ASP.NET Core)
* PostgreSQL
* Swagger
* Dapper
* BCrypt

O sistema permite:

* Cadastro de usuários
* Login com autenticação
* Criptografia de senhas
* Integração com banco de dados PostgreSQL
* Testes de API via Swagger
